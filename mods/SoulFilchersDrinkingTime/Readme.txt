Soul Filcher's Drinking Time mod
By Soul Filcher, 2021.
Downloaded from Steam Workshop.

More items can be used to hold water.


-----MODIFIED ITEMS----------------------------------------------------------------

-Jar (Empty):          Can now be used to hold water.
-Paint Bucket (Empty): Can now be used to hold and boil water.
-Pop Can (Empty):      Can now be used to hold water.
-Tin Can (Empty):      Can now be used to hold and boil water.
-Water Dish:           Can now be used to hold water.
